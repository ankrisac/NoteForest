# NoteForest

![](https://github.com/ankrisac/NoteForest/blob/main/doc/example.png)

## Controls
Ctrl-Y : Toggle viewer
Ctrl-E : Toggle editor

## Features
- typesetting DSL
- LaTeX typesetting support (via Katex)
- Electrical Circuit & Diagram rendering DSL
